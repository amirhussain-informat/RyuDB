Here's a SQL-surface comparison between DuckDB and RyuDB, based on what RyuDB actually implements today (master + the
  two unmerged feature branches). I've kept it honest about RyuDB's gaps.

  DDL

  ┌──────────────────────────────┬────────┬─────────────────────────────────────────────────────────────────────────┐
  │             Area             │ DuckDB │                                  RyuDB                                  │
  ├──────────────────────────────┼────────┼─────────────────────────────────────────────────────────────────────────┤
  │ CREATE TABLE (typed, with    │ Full   │ CREATE TABLE ... FROM '<parquet>'; constraints via :alter (NOT          │
  │ constraints)                 │        │ NULL/PK/UNIQUE/DEFAULT stored, enforced on write)                       │
  ├──────────────────────────────┼────────┼─────────────────────────────────────────────────────────────────────────┤
  │ ALTER TABLE (add/drop/alter  │ Full   │ Rename/drop table; constraint set/unset only — no add/drop/modify       │
  │ col, rename)                 │        │ column                                                                  │
  ├──────────────────────────────┼────────┼─────────────────────────────────────────────────────────────────────────┤
  │ DROP TABLE                   │ Full   │ :drop / drop_table                                                      │
  ├──────────────────────────────┼────────┼─────────────────────────────────────────────────────────────────────────┤
  │ Views (CREATE VIEW)          │ Full   │ Not supported                                                           │
  ├──────────────────────────────┼────────┼─────────────────────────────────────────────────────────────────────────┤
  │ Schemas/databases            │ Full   │ Single catalog (<data_dir>/ryudb.catalog.json)                          │
  ├──────────────────────────────┼────────┼─────────────────────────────────────────────────────────────────────────┤
  │ Indexes                      │ Full   │ None (GPU scans; PK used for tombstone anti-join, not an index)         │
  └──────────────────────────────┴────────┴─────────────────────────────────────────────────────────────────────────┘

  DML

  ┌─────────────────────────────┬────────┬──────────────────────────────────────────────────────────────────────────┐
  │            Area             │ DuckDB │                                  RyuDB                                   │
  ├─────────────────────────────┼────────┼──────────────────────────────────────────────────────────────────────────┤
  │ INSERT … VALUES             │ Full   │ Full (typed coercion, NOT NULL/DEFAULT/PK/UNIQUE enforced)               │
  ├─────────────────────────────┼────────┼──────────────────────────────────────────────────────────────────────────┤
  │ INSERT … SELECT             │ Full   │ Full (branch feature/insert-select, unmerged)                            │
  ├─────────────────────────────┼────────┼──────────────────────────────────────────────────────────────────────────┤
  │ UPDATE                      │ Full   │ Full (PK-tombstone + re-insert)                                          │
  ├─────────────────────────────┼────────┼──────────────────────────────────────────────────────────────────────────┤
  │ DELETE … WHERE              │ Full   │ Full (requires declared PK)                                              │
  ├─────────────────────────────┼────────┼──────────────────────────────────────────────────────────────────────────┤
  │ ON CONFLICT / upsert /      │ Full   │ Not supported                                                            │
  │ MERGE                       │        │                                                                          │
  ├─────────────────────────────┼────────┼──────────────────────────────────────────────────────────────────────────┤
  │ RETURNING                   │ Full   │ Not supported                                                            │
  ├─────────────────────────────┼────────┼──────────────────────────────────────────────────────────────────────────┤
  │ COPY/import/export          │ Full   │ Reads Parquet via FROM; no COPY TO SQL (checkpoint writes Parquet        │
  │                             │        │ internally)                                                              │
  └─────────────────────────────┴────────┴──────────────────────────────────────────────────────────────────────────┘

  Query — projection & filtering

  ┌────────────────────────────────────────────────┬────────────────────┬──────────────────────────────────────────┐
  │                      Area                      │       DuckDB       │                  RyuDB                   │
  ├────────────────────────────────────────────────┼────────────────────┼──────────────────────────────────────────┤
  │ SELECT cols / * / aliases                      │ Full               │ Full                                     │
  ├────────────────────────────────────────────────┼────────────────────┼──────────────────────────────────────────┤
  │ WHERE (predicates, AND/OR/NOT, 3-valued logic) │ Full               │ Full                                     │
  ├────────────────────────────────────────────────┼────────────────────┼──────────────────────────────────────────┤
  │ DISTINCT / DISTINCT ON                         │ Both               │ DISTINCT only; DISTINCT ON not supported │
  ├────────────────────────────────────────────────┼────────────────────┼──────────────────────────────────────────┤
  │ ORDER BY                                       │ Full (expressions) │ Column refs only (no expressions)        │
  ├────────────────────────────────────────────────┼────────────────────┼──────────────────────────────────────────┤
  │ LIMIT / OFFSET                                 │ Full               │ Full                                     │
  ├────────────────────────────────────────────────┼────────────────────┼──────────────────────────────────────────┤
  │ CASE / COALESCE / CAST / NULLIF                │ Full               │ CASE/COALESCE/CAST; no NULLIF            │
  └────────────────────────────────────────────────┴────────────────────┴──────────────────────────────────────────┘

  Joins

  ┌─────────────────────────────────────────────┬────────┬────────────────────────────────────────────────────────┐
  │                    Area                     │ DuckDB │                         RyuDB                          │
  ├─────────────────────────────────────────────┼────────┼────────────────────────────────────────────────────────┤
  │ Inner / equi-join                           │ Full   │ Full                                                   │
  ├─────────────────────────────────────────────┼────────┼────────────────────────────────────────────────────────┤
  │ LEFT / RIGHT / FULL OUTER                   │ Full   │ Full                                                   │
  ├─────────────────────────────────────────────┼────────┼────────────────────────────────────────────────────────┤
  │ CROSS / NATURAL                             │ Full   │ Full                                                   │
  ├─────────────────────────────────────────────┼────────┼────────────────────────────────────────────────────────┤
  │ Non-equi joins (>, !=, arbitrary predicate) │ Full   │ Equi-join focused; non-equi via ON predicate supported │
  ├─────────────────────────────────────────────┼────────┼────────────────────────────────────────────────────────┤
  │ LATERAL                                     │ Full   │ Not supported                                          │
  ├─────────────────────────────────────────────┼────────┼────────────────────────────────────────────────────────┤
  │ Semi/anti (via IN/EXISTS)                   │ Full   │ Lowered to semi/anti join                              │
  └─────────────────────────────────────────────┴────────┴────────────────────────────────────────────────────────┘

  Grouping & aggregates

  ┌───────────────────────────┬───────────────────────────────────────────────┬─────────────────────────────────────┐
  │           Area            │                    DuckDB                     │                RyuDB                │
  ├───────────────────────────┼───────────────────────────────────────────────┼─────────────────────────────────────┤
  │ GROUP BY                  │ Full                                          │ Full (incl. no-agg SELECT a … GROUP │
  │                           │                                               │  BY a, now fixed)                   │
  ├───────────────────────────┼───────────────────────────────────────────────┼─────────────────────────────────────┤
  │                           │ Huge library (SUM/AVG/COUNT/MIN/MAX, STDDEV,  │                                     │
  │ Aggregates                │ VARIANCE, MEDIAN, PERCENTILE, ARRAY_AGG,      │ Only COUNT/SUM/AVG/MIN/MAX          │
  │                           │ STRING_AGG, BOOL_AND/OR, …)                   │                                     │
  ├───────────────────────────┼───────────────────────────────────────────────┼─────────────────────────────────────┤
  │ GROUP BY expressions      │                                               │ Full for flat GROUP BY; not in      │
  │ (e.g. a%10,               │ Full                                          │ ROLLUP/CUBE/GROUPING SETS (bare     │
  │ date_trunc(...))          │                                               │ columns only)                       │
  ├───────────────────────────┼───────────────────────────────────────────────┼─────────────────────────────────────┤
  │ HAVING                    │ Full                                          │ Full; not with ROLLUP/CUBE/GROUPING │
  │                           │                                               │  SETS (deferred)                    │
  ├───────────────────────────┼───────────────────────────────────────────────┼─────────────────────────────────────┤
  │ ROLLUP / CUBE / GROUPING  │ Full                                          │ Full (branch feature/grouping-sets, │
  │ SETS                      │                                               │  unmerged)                          │
  ├───────────────────────────┼───────────────────────────────────────────────┼─────────────────────────────────────┤
  │ GROUPING()                │ Full                                          │ Full (single + multi-arg bitmap)    │
  ├───────────────────────────┼───────────────────────────────────────────────┼─────────────────────────────────────┤
  │ Composite grouping items  │ Full                                          │ Full                                │
  │ (ROLLUP(a,(b,c)))         │                                               │                                     │
  ├───────────────────────────┼───────────────────────────────────────────────┼─────────────────────────────────────┤
  │ FILTER clause on          │                                               │                                     │
  │ aggregates (SUM(x) FILTER │ Full                                          │ Not supported                       │
  │  (WHERE …))               │                                               │                                     │
  └───────────────────────────┴───────────────────────────────────────────────┴─────────────────────────────────────┘

  Set operations

  ├────────────────────────────┼────────┼──────────────────────┤

  Subqueries

  Transactions & durability

  ┌───────────────────────────────┬─────────────────┬───────────────────────────────────────────────────────────────┐

  ├───────────────────────────────┼─────────────────┼───────────────────────────────────────────────────────────────┤
  │ BEGIN/COMMIT/ROLLBACK         │ Full (ACID,     │ Full (MVCC, read-your-writes, atomic commit)                  │
  │                               │ MVCC)           │                                                               │
  ├───────────────────────────────┼─────────────────┼───────────────────────────────────────────────────────────────┤
  │ Savepoints                    │ Full            │ Not supported                                                 │
  ├───────────────────────────────┼─────────────────┼───────────────────────────────────────────────────────────────┤
  │ Snapshots / RESTORE TO        │ (via backups)   │ In-memory named snapshots + restore_to(ts). Not persistent    │
  │ SNAPSHOT                      │                 │ across restart                                                │
  ├───────────────────────────────┼─────────────────┼───────────────────────────────────────────────────────────────┤

  ├───────────────────────────────┼─────────────────┼───────────────────────────────────────────────────────────────┤
  │ Checkpoint (base write-back)  │ Full            │ Full (DECIMAL/DATE fidelity on disk)                          │
  ├───────────────────────────────┼─────────────────┼───────────────────────────────────────────────────────────────┤

  Execution / storage model (where RyuDB differs by design)

  ┌────────────┬──────────────────────────────┬────────────────────────────────────────────────────────────────────┐
  │   Aspect   │            DuckDB            │                               RyuDB                                │
  ├────────────┼──────────────────────────────┼────────────────────────────────────────────────────────────────────┤
  │ Engine     │ CPU vectorized (row-group,   │ GPU/cuDF + C++/nvcc fused kernels                                  │
  │            │ morsel-driven)               │                                                                    │
  ├────────────┼──────────────────────────────┼────────────────────────────────────────────────────────────────────┤
  │ HTAP       │ OLAP-focused                 │ OLAP scan/agg + OLTP write path (delta + WAL + MVCC)               │
  ├────────────┼──────────────────────────────┼────────────────────────────────────────────────────────────────────┤
  │ Cold path  │ Columnar Parquet reader      │ Hand-rolled Parquet reader (nvCOMP) — DEFAULT cold path; cold wall │
  │            │                              │  ≈ H2D bytes (~80ms, PCIe 6.6 GB/s WSL2)                           │
  ├────────────┼──────────────────────────────┼────────────────────────────────────────────────────────────────────┤
  │ Fused      │ n/a                          │ global agg + MIN/MAX/AVG + fused star-join+aggregate               │
  │ kernels    │                              │                                                                    │
  ├────────────┼──────────────────────────────┼────────────────────────────────────────────────────────────────────┤
  │ Perf at    │ Faster end-to-end            │ Does not yet beat DuckDB end-to-end (0.10×–0.45×); beats DuckDB    │
  │ SF10       │                              │ warm on 6/6 bench kernels                                          │
  └────────────┴──────────────────────────────┴────────────────────────────────────────────────────────────────────┘
